### Задание 2.3: Настройка chrony как NTP-сервера для локальной сети

Откройте конфигурационный файл chrony для редактирования:

```sh
sudo nano /etc/chrony.conf
```

Добавьте следующую строку в конфигурационный файл, чтобы разрешить синхронизацию времени для сети 10.0.10.0/24:

```sh
allow 10.0.10.0/24
```

**Примечание:** Эта директива должна быть добавлена в раздел с комментарием "Allow NTP client access from local network" или в конец файла.

Сохраните изменения (в nano: Ctrl+O, Enter, Ctrl+X).

### Задание 2.6: Настройка firewall для NTP-сервера

Откройте порт 123/udp в firewall для доступа клиентов из локальной сети:

```sh
sudo firewall-cmd --add-service=ntp --permanent
sudo firewall-cmd --reload
```
